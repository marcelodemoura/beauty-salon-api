# Beauty Salon API

API para gestão de salão de beleza, incluindo:
- Serviços
- Clientes
- Profissionais
- Agendamentos
- Preços
- Pagamentos (futuro)

Tecnologias:
- Spring Boot 3
- Java 17
- PostgreSQL
- JPA/Hibernate
